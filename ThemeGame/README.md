Задача из темы "Игры".

Условие:

В игре на орграфе два игрока поочередно накрывают фиш­ками вершины орграфа: один белыми, а другой черными. Игрок при своем ходе может накрыть фишкой любую из свободных вершин, хотя бы один предшественник которой накрыт фишкой противника; первым ходом белые накрывают любую вершину. Проигрывает тот, который при своем ходе не может выставить фишку в соответствии с правила­ми игры. Определить, является ли начальная конфигурация игры на заданном орграфе выигрышной для белых.
