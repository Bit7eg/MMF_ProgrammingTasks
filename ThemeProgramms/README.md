Задача из темы "Формулы и программы".

Условие:

Необходимо написать синтаксический анализатор для логических формул.

Описание программы:

Пользователь вводит строку. Программа определяет является ли введённая строка логической формулой. Если строка является логической формулой, то программа выводит "It is logical formula", иначе программа выводит "It isn't logical formula"

Дополнительные файлы:

LogicalFragmentAnaliserFunctions.h - объявление функций для определения логической формулы и её частей.
LogicalFragmentAnaliserFunctions.c - описание функций для определения логической формулы и её частей.
LogicalFragmentGeneratorFunctions.h - объявление функций для генерирования логической формулы и её частей.
LogicalFragmentGeneratorFunctions.c - описание функций для генерирования логической формулы и её частей.
StringLib.h - объявление структуры и функций для удобной работы со строками.
StringLib.c - описание функций для удобной работы со строками.
