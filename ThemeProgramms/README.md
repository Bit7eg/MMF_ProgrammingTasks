Задача из темы "Формулы и программы".

Условие:

Определить, эквивалентны ли два заданных логических фрагмента.

Описание программы:

Пользователь задаёт два логических фрагмента. Фрагменты задаются по очереди, каждая вершина описывается в отдельной строчке в формате "N P M", где N - имя вершины, не являющейся листком, P - имя вершины или номер листка, в которые происходит переход по плюс-стрелке, M - имя вершины или номер листка, в которые происходит переход по минус-стрелке. Имя вершины, не являющейся листком, это заглавная буква латинского алфавита. Для завершения ввода логического фрагмента необходимо ввести строку "- - -". После ввода логических фрагментов, взависимости от результата работы, программа выводит одну из трх строк: "Equal", если логические фрагменты эквивалентны, "Not equal", если логические фрагменты неэквивалентны, и "Incorrect data", если хотя бы один из введнных логических фрагментов не является логическим фрагментом.

Дополнительные файлы:

BinGraphLib.h - объявление структуры логического фрагмента и функций для работы с ней.
BinGraphLib.c - описание функций для работы с логическим фрагментом.
ValueSetLib.h - объявление структуры множества значений именованных вершин и функций для работы с ней.
ValueSetLib.c - описание функций для работы с множеством значений именованных вершин.
