Задача из темы "Графы".

Условие:

Для заданного натурального n >= 3 построить граф, не содержащий циклов длиной 3, в котором степени всех вершин равны 3.

Описание программы:

Пользователь вводит число. Программа строит граф, не содержащий циклов длиной 3, в котором степени всех вершин равны 3, и выводит матрицу смежности построенного графа.

Дополнительные файлы:

GraphLib.h - описание типа GRAPH и объявление функций для работы с ним.
GraphLib.c - описание функций для для работы с типом GRAPH.
